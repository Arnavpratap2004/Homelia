# Catalogues Folder

This folder contains the PDF catalogues for both laminate brands.

## File Naming Convention

Please add your PDF files with the following names:

- `durian-catalogue.pdf` - Durian Laminates full catalogue
- `rockstar-catalogue.pdf` - Rockstar Laminates full catalogue

## Usage

These PDFs will be accessible at:
- `/catalogues/durian-catalogue.pdf`
- `/catalogues/rockstar-catalogue.pdf`

The brand pages and product pages link to these catalogues for download.
